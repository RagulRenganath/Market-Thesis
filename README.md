# Market Thesis - Institutional Research Platform

Market Thesis is a modern, frontend web platform designed to deliver sharp, research-driven perspectives on Indian markets, business, and finance. The platform translates complex macroeconomic developments into clear, qualitative viewpoints tailored for professionals and institutional decision-makers. 

## Technical Architecture
This project is built utilizing a lightweight, highly responsive frontend stack consisting of vanilla HTML5, CSS3, and JavaScript. The design system employs a modern light-mode interface with glassmorphism elements and smooth scroll-reveal animations, ensuring maximum readability and an enterprise-grade feel for long-form financial research. 

## Core Integrations
The platform incorporates live data streams and interactive features without relying on a heavy backend server or paid API keys. The live market terminal utilizes the TradingView advanced widget to track real-time movements in the NIFTY 50, SENSEX, and key macroeconomic indicators flawlessly. 

To ensure the content remains dynamic, the site integrates an automated data stream via rss2json, which bypasses browser CORS restrictions to fetch and display live Indian financial news directly from the Economic Times RSS feed. Additionally, the platform features a custom-built, Typeform-style commission portal. This portal uses FormSubmit to route custom research requests directly to the analyst's email, creating a seamless and secure user experience.

## Deployment
The application is designed for immediate deployment via GitHub Pages. Because it relies entirely on client-side rendering and external endpoints for dynamic content, it functions as a highly capable static site while still providing real-time financial data.
